# Aplicación Web de Análisis de Vehículos

## Descripción del proyecto

Esta aplicación web fue desarrollada utilizando Streamlit y Python con el objetivo de analizar un conjunto de datos de anuncios de venta de vehículos en Estados Unidos.

La aplicación permite visualizar y explorar información relevante sobre los automóviles mediante gráficos interactivos y tablas dinámicas, facilitando el análisis de variables como precio, kilometraje y condición del vehículo.

## Funcionalidades principales

- Visualización de histogramas para analizar la distribución del kilometraje de los vehículos.
- Gráficas de dispersión para observar la relación entre el odómetro y el precio.
- Comparación entre la condición del vehículo y su precio mediante gráficos tipo boxplot.
- Visualización interactiva del dataframe con los datos de los vehículos.
- Interfaz interactiva desarrollada con Streamlit.

## Tecnologías utilizadas

- Python
- Pandas
- Plotly
- Streamlit
